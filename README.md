## installation
```bash
npm install -S @feizheng/next-stub
```

## apis
| api           | description               |
|---------------|---------------------------|
| nx.stubString | Stub empty and new string |
| nx.stubArray  | Stub empty and new array  |
| nx.stubObject | Stub empty and new object |
| nx.stubThen   | Stub empty and new then   |
